Developer Profile

So, Linda - What was your first introduction to web development?

I found myself in a position where I wasn't satisfied with my previous work experience, and I had time to teach myself something new. I didn't have the funds to go to a traditional school, but knew I could find resources online. And I came across Harvard's Intro to Computer Science offering on the edX website, CS50. It sounded difficult and interesting, and since I could audit the course for free, I figured, why not? So I signed up. And it was difficult, and often frustrating. But it was also fun, and interesting, and I made it through and realized that I wanted to learn and create more.

What languages have you worked with so far? Which ones have you been most interested in?

Well, CS50 introduced me to C, HTML, CSS, and PHP, with a little bit of MySQL as well. Through Free Code Camp, I'm using mostly HTML, CSS, and JS right now.
The web side of things has held my interest the most, so I'm really focusing on HTML, CSS, and JS at the moment, and I want to learn more about SQL and other database languages in the future, as I love collecting and utilizing data.

Do you have a code editor of choice?

Currently I've been using Visual Studio Code, and am really enjoying it. I've also used Cloud 9 (which is what we used for CS50) and Notepad++.
