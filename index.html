Projects

This task asked us to create a weather app where you could see your local weather, have an icon that changes depending on the weather conditions, and toggle between Fahrenheit and Celcius. This is my first program that gathered information from an API. Since we are using Codepen for our code, there was an extra little hoop to jump through to return the data (cross origin/https technicality), but I managed to get it working without too much trouble. The thing that honestly gave me the most difficulty with this task was the weather icons. I found a set that I liked better than the ones that came with the Open Weather Map API, and it sounded easy to set up. Famous last words, right? It took a lot of tweaking and frustration, and it's still not as clean/perfect as I would like (the spacing is still off and the layered icons don't always line up exactly how I'd like), but it's a lot closer to what I had in mind. The background colors of the labels also change depending on the time of day and the type of the weather. This project is viewable here.

For the Free Code Camp curriculum, we were tasked with creating a random quote generator. I created a list in another pen of quotes that I wanted to use, and linked that pen to my generator to cycle through the quotes. This task helped me learn more about Javascript functions, including setting up a way to cycle through quotes and colors, ensuring selections weren't immediately repeated, and setting up a Twitter button and how to select where and how a post would end if it was too long for a tweet. It also gave me an inital idea of how to access information from a different source. This project is viewable here.

This project came about because I wanted a number pad for a form. I track delivery locations and invoice numbers in my current position, and I had been using a Google form to input this information into a spreadsheet. However, the form didn't have much I could really customize. By creating my own form, I could get the number keypad I so badly wanted, I could stay on the same page once I had entered the information (using a small alert to show it had been logged into the spreadsheet), and I could control the look more.

Logolinks was my final project for CS50, Harvard's intro to computer science class given through edX. My partner had told me he'd always wanted a place where he could store links for easy access, similar to bookmarks but a different way of viewing them, where it was easier to copy the links and share them with others. This was what I came up with. This project is viewable here.

For my first project, I created a business card order form for a company client. They had been sending their orders to us through fax, and had requested a way to be able to email us the information. I volunteered myself and created this, which emails the form information to our sales department. They have been very pleased with it, simple as it is.
